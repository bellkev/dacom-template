dacom-template
====

A Leiningen template for the DACOM (**Da**tomic, **C**ompojure, **Om**) stack.

## Usage

```
lein new dacom my-dacom-app
```
(Requires Leiningen 2.X)

See [the project page](https://github.com/bellkev/dacom) for more info.
